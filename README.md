# 點餐系統
這是一個使用 C# Windows Forms 開發的點餐系統，並搭配內建 SQL 資料庫。  

## 功能特色
1. **帳號與密碼登入**
    - 使用 SHA256 加密驗證
    - 支援一般帳號（桌次）與管理員帳號 (root)
2. **多視窗設計**
    - 登入畫面
    - 點餐畫面
    - 點餐清單畫面
3. **菜單管理**
    - 菜單存放於資料庫
    - 動態載入餐點名稱、圖片與價格
4. **訂單功能**
    - 新增餐點至清單並修改數量
    - 送出訂單後將資料寫入資料庫

---

## 使用技術
- C#
- Windows Forms
- SHA256 加密
- SQL Server (內建資料庫)
